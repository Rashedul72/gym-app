# Workout exercise images

Copy your PNG files from **gym-web** into this folder.

## Source (web app)

```
gym-web/public/images/workouts/
```

## Destination (mobile app)

```
my-gym-mobile/assets/images/workouts/
```

Use the **same filenames** as the web app. After copying, uncomment the matching lines in `lib/imageRegistry.ts` (or ask the agent to enable them once files are present).
